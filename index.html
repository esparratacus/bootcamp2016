Funciones JS

Una funcion es un bloque de codigo que se ejecuta cuando lo llamamos. Una funcion tiene un numero de entradas dependiendo del problema a resolver que se usaran dentro de dicho bloque de codigo para solucionar un problema en particular. Tambien puede tene una salida con la solucion al problema en cuestion.

Definicion de una funcion

Una funcion se puede almacenar en una variable de la siguiente manera:

var miFuncion= function(entrada1,entrada2)
{

}

Este caso nuestra funcion se llama miFuncion y tiene como entradas entrada1 y entrada2.

Cuerpo de una funcion

Hasta el momento nuestra funcion no tiene ningun proposito. El cuerpo de una funcion es el bloque de codigo que queremos que se ejecute cuando vayamos a usarla.

var areaRectangulo= function(ancho,alto){
    return ancho*alto;
}

Esta funcion llamada areaRectangulo tiene como entradas (tambien llamamdas argumentos o parametros) el alto y el ancho de un ractangulo y en el cuerpo de la funcion hacemos los calculos pertinentes y retornamos el area con la palabra reservada de JavaScript return. El cuerpo de una funcion esta delimitado entonces por las llaves {}

Llamados a funciones

Hasta el momentos hemos definido nuestra funcion. Si ejecutamos este codigo no tendremos ningun error. Sin embargo hasta el momento no hemos hecho uso de la funcion, simplemente la hemos definido. Esto quiere decir que hasta el momento Javascript ha leido nuestro codigo y sabe que existe dicha funcion pero no ha visto ninguna ejecucion de la misma. Para lograr esto tenemos que hacer uso de ella o "llamarla".

var lado1=3;
var lado2=5;
var area;
area= areaRectangulo(lado1,lado2);
console.log(area);
//en la consola se imprimira 15
  1. Creamos 2 variables donde tendremos el tamanio de cada lado respectivamente.
  2. Declaramos una tercera variable donde almacenaremos el area de nuestro rectangulo
  3. A la variable area le asignamos el valor retornado por la funcion areaRectangulo LLAMANDO a la funcion areaRectangulo insertando lado1 y lado2 como entrada o parametros
  4. Imprimimos por consola el resultado

Arreglos JS

Que es un arreglo?

Hasta el momento hemos declarado variables donde podemos almacenar un solo dato en dicha variable. Sin embargo nos encontraremos en la necesidad de guardar varios datos en una sola variable para poder manejarlos despues. Para estas situaciones usaremos arreglos o arrays para hacer frente a esta situacion

Declaracion de arreglos

Para crear un arreglo vacio basta con asignar a cualquier variable parentesis cuadrados[]

var arreglo = [];

Si queremos crear un arreglo y agregarle elementos al mismo tiempo podemos hacerlo de la siguiente manera:

var arreglo = ["papaya","mango","naranja"];

Es importante recalcar que los arreglos pueden contener CUALQUIER de dato incluso otro arreglo

Acceso a elementos de un arreglo

Al tener un arreglo creado y con elementos es logico que en algun momento de nuestro programa queramos acceder a ellos. Antes de entrar en detalle acerca de como acceder a un elemento de un arreglo primero es importante entender los terminos que lo componen.

  • tamanio: El numero total de elementos que tiene el arreglo
  • indice: Es la posicion en la cual esta un elemento en el arreglo contando desde 0 en la primera posicion

Importante! Es importante tener en cuenta la diferencia entre indice y tamanio! El numero total de indices de un arreglo es el tamanio del arreglo menos 1. ej: Si el tamanio del arreglo es 10 las posiciones posiblee en dicho arreglo van de 0 a 9.

Ahora si retomamos nuestro ejemplo de arreglo de frutas este tiene un tamanio de 3 por lo que las posiciones posibles en las cuales podemos situarnos en el arreglo es de 0 a 2. La posicion 0 del arreglo corresponde a la palabra papaya, la posicion 1 a la palabra mango y la posicion 2 a la palabra naranja.

Para acceder a uno de estos elementos lo hacemos a traves de su indice de la siguiente manera:

var arreglo = ["papaya","mango","naranja"];
var fruta = arreglo[0];
console.log(fruta);
//imprime papaya
fruta = arreglo[1];
console.log(fruta);
//imprime mango
fruta = arreglo[2]
console.log(fruta);
//imprime naranja

Lo que hicimos fue declarar una variable nueva llamada fruta y le asignamos el valor de arreglo en el indice demarcado por [].

Agregar elementos a un arreglo

Para agregar elementos a un arreglo, JS tiene funciones que nos permiten realizar esta operacion facilmente. Entre estas se encuentra la funcion push que agrega un elemento al principio del arreglo.

var arreglo = ["papaya","mango","naranja"];
arreglo.push("manzana");
console.log(arreglo);
// arreglo es ahora ["papaya","mango","naranja","manzana"]

Iteradores

Concepto

Hay veces que requerimos que nuestro codigo se ejecute varias veces para hacer la misma operación. Por ejemplo, si estamos manjenado nuestro arreglo de frutas y queremos imprimir el nombre de dicha fruta tendríamos que hacer los siguiente:

var arreglo = ["papaya","mango","naranja"];
console.log(arrglo[0]);
console.log(arrglo[1]);
console.log(arrglo[2]);

Ahora qué pasaría si nuestro arreglo no tuiera simplemente 3 frutos sino cientos de ella? Sería bastante dispendioso escribir la misma operación cientos de veces. Aqui es donde los iteradores facilitan bastante la vida del programador. Si usamos iteradores para para el ejemplo anterior tendríamos lo siguiente:

var arreglo = ["papaya","mango","naranja"];
for(var i = 0;i<arreglo.length;i++){
   console.log(arreglo[i]);
}

Básicamente lo que estamos haciendo con este código en palabras informales es:

"Empezando desde 0 hasta el final del arreglo, tome cada elemento e imprímalo"
